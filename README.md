# Cat VS Dog CHESS

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Status: In Development](https://img.shields.io/badge/Status-In--Development-orange)

Ein klassisches Schachspiel mit einem animalischen Twist. In CatVSDogCHESS uebernehmen Katzen und Hunde das Brett. Wer wird die Oberhand gewinnen: Die strategischen Samtpfoten oder die treuen Vierbeiner?

---

## Projekt-Idee

Dieses Projekt kombiniert die zeitlose Logik von Schach mit einem individuellen Design. Es dient als modernes Beispiel fuer eine Spiele-App, die auf klarem Code (TypeScript) und einer funktionalen UI basiert.

### Features
* Custom Pieces: Katzen (Weiss) gegen Hunde (Schwarz).
* Intuitive Steuerung: Einfaches Klicken und Ziehen der Figuren.
* Schachregeln: Komplette Implementierung inklusive En Passant, Rochade und Bauerumwandlung.
